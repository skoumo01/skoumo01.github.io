  • ASM4DEAF

    Project Title: Accessible System and Social Media Mobile application For Deaf users (ASM4Deaf)

    Project Website: www.asm4deaf.eu

    Project Description: The output of the project will be a GIF keyboard specifically created for deaf users. The GIFs will be in American Sign Language and the users will have the possiblity to have fun creating their own versions of the (original) GIFs they want to use (personalize them).

    Assigned Tasks:

    • Aid in technical research and requirements extraction.
    • Design and implemenation of GIFs' database.
    • Coding of Flask (Python) REST API for GIF management (add, query, retrieve, edit, delete).
    • Coding of a Flask (Python) web Graphical User Interface for database management and API testing.
    • Creation of WordPress website.
